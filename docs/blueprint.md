# **App Name**: Vibrant Portfolio

## Core Features:

- Smooth Section Navigation: Implement smooth scrolling for navigation between portfolio sections.
- Interactive Project Gallery: Create a gallery with project images and animated hover effects. Utilize a tool that dynamically rearranges and presents portfolio projects, considering visual appeal and user engagement metrics.
- Dynamic 'About Me' Section: Develop a dynamic section highlighting skills and experience with animated transitions.
- Integrated Contact Form: Integrate a functional contact form with Tailwind CSS styling for user inquiries.

## Style Guidelines:

- Background color: Dark grey (#222222) for a modern, sophisticated feel.
- Primary color: Electric blue (#7DF9FF) to draw attention to the project highlights.
- Accent color: Deep purple (#A020F0) for interactive elements and hover states.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern aesthetic. Note: currently only Google Fonts are supported.
- Use minimalist, custom-designed icons for navigation and section headers.
- Implement a responsive grid layout with Tailwind CSS for optimal viewing on all devices.
- Incorporate subtle, tasteful animations for section transitions and element interactions.
